Launch Instructions:  
Install python 3.1x and clone the repository  
Get a google books API key from https://console.cloud.google.com and modify GOOGLE_BOOKS_API_KEY in app.py to be the API key  
Run the following commands in the terminal in the directory containing app.py:  
pip install -r requirements.txt  
py app.py
